# redstone2D

Только с пк: https://rfldxx.github.io/redstone2D/

Управление:
- Выбор из инвентаря: цифра 1-5 или ЛКМ по иконке
- Изменение порядка иконок в инвентаре: зажать ЛКМ на иконке (до заполнения шкалы) и затем drag&drop в нужную позицию
- Вращение текущего предмета: r
- Размещение текущего предмета: ЛКМ на поле
- Копирование: клик колёсиком мышки по предмету (для копирования микросхемы надо кликнуть по рамке микросхемы)
- Перетаскивание микросхемы: drag&drop (по рамке микросхемы) через зажатие ПКМ
- Область вставки микросхемы подсвечивается. Чтобы микросхема перетерла все предметы в области вставки - зажать ctrl
